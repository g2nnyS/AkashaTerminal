<h1 align="center">虚空终端</h1>

<p align="center">基于<a href="https://github.com/huashengdun/webssh">WebSSH</a>修改的一款SSH工具，灵感来源于原神中的虚空终端</p>

## 开发与调试
打开cmd或Shell,运行以下指令:
```
pip install Pyqt5
```
```
pip install WebSSH
```

随后点击run.py运行
